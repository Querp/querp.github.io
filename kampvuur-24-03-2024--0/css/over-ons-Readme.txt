Als deze pagina niet in dezelfde folder neergezet wordt als de image en de stylesheet, vergeet alstublieft niet om de href/src aan te passen.

Je kan meerdere css styles in een pagina laden. Ik heb het stukje css wat alleen relevant is voor deze pagina in een aparte css gedaan (genaamd overons-styles.css).
Voor de rest van de layout kan deze pagina ook een styles.css laden die in dezelfde map staat. Als hij in een andere map staat, moet de href in <head> aangepast worden.
